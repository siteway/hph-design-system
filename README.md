# HPH Design System

Das offizielle Design System von **Homepage Helden GmbH** — gebaut mit [Fractal](https://fractal.build/).

## 🔗 Live

👉 **[design.homepage-helden.de](https://siteway.github.io/hph-design-system)**

## 🛠️ Lokal starten

```bash
npm install
npm start
# → http://localhost:3000
```

## 📦 Build

```bash
npm run build
# → /build
```

## 📐 Was ist drin?

- **Tokens** — Farben, Typografie, Spacing, Gradients
- **Komponenten** — Buttons, Text, Cards, alle Page-Builder-Module
- **Dokumentation** — Nutzungshinweise, Inkonsistenzen, Changelog

## 🔄 Auto-Deploy

Jeder Push auf `main` deployed automatisch via GitHub Actions auf GitHub Pages.

## 🏗️ Stack

- [Fractal](https://fractal.build/) — Component Library Tool
- [Tailwind CSS](https://tailwindcss.com/) — Styling
- [Inter](https://fonts.google.com/specimen/Inter) — Schrift

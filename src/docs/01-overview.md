# HPH Design System

Das Design System von **Homepage Helden** ist die zentrale Referenz für alle visuellen und strukturellen Entscheidungen.

## Wozu dient dieses System?

- **Konsistenz** — Gleiche Tokens in Figma, Tailwind-Config und Templates
- **Geschwindigkeit** — Neue Seiten aus bestehenden Modulen zusammenbauen
- **Dokumentation** — Für Entwickler, Designer und neue Team-Mitglieder

## Struktur

```
Tokens       → Farben, Typografie, Spacing, Gradients
Komponenten  → UI-Elemente (Buttons, Cards, ...)
Module       → Page-Builder-Blöcke (Text, Hero, Logo Wall, ...)
```

## Technologie

| Layer | Tool |
|---|---|
| Design | Figma (HPH Relaunch 2023) |
| Styling | Tailwind CSS 3 |
| Templates | Antlers (Statamic 5) |
| CMS | Statamic auf Laravel 12 |
| Font | Inter + Cera Variable |

| Media | Bild oder Video-Element |
| Quote Box | Hervorgehobenes Zitat |
| Sitemap | Automatische Seitenübersicht |
| Tabs | Inhalte in Tab-Navigation |
| Text Media | Text + Bild (positionierbar) |

| Code | Syntax-Highlighting Code-Block |
| Teaser Cards | Vorschau-Karten (z.B. für Blog) |

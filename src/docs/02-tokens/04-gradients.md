# Gradients

## Hero Gradient (Hell)
```css
background: linear-gradient(72.87deg, #1548BB 27.5%, #027CEC 85.64%);
```
Klasse: `bg-hero-gradient`

## Hero Gradient (Dunkel)
```css
background: linear-gradient(123deg, #000 0%, #1F1F1F 100%);
```
Klasse: `bg-hero-gradient-dark`

## Button Gradient
```css
background: linear-gradient(90.79deg, #3376F6 0.46%, #0040B8 99.09%);
```
Klasse: `bg-button-gradient`

## Headline Gradient (Text)
```css
background: linear-gradient(71.08deg, #003EC8 30.98%, #67BAEF 76.86%);
-webkit-background-clip: text;
```
Klasse: `bg-headline-gradient bg-clip-text text-transparent`
